Detailed Description

This is an abtstract class that calculates a coordinate (x,y) from an index (iw,ih) for a tile. Any derived class need not be cartesian but could also be polar.

USAGE: Before the calc_... methods can be called, the coordinate system has to be initialized with the given tile using the init(...) method. So the object itself just lays the rules for how to calculate a coordinate (x,y) from a given index (iw,ih).
