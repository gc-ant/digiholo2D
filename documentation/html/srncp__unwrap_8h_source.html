79 //this function tries to implement a nice idea explained below
80 //we need to sort edge array. Each edge element conisists of 16 bytes.
81 //In normal sort program we compare two elements in the array and exchange
82 //their place under some conditions to do the sorting. It is very probable
83 // that an edge element may change its place hundred of times which makes
84 //the sorting a very time consuming operation. The idea in this function
85 //is to give each edge element an index and move the index not the edge
86 //element. The edge need 4 bytes which makes the sorting operation faster.
87 // After finishingthe sorting of the indexes, we know the position of each index.
88 //So we know how to sort edges
