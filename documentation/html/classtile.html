Detailed Description

Unterteilung eines Bildes in Kacheln (tiles). Die Daten der Tiles sind nicht hintereinander im Speicher abgelegt, darum kann sie auch nicht von float_image abgeleitet sein.

This methods allcocates memory for a 1D Array and copies the data that this tile points to into this array in a row (width) major fashion. The pointer to the copy of the data has to be deleted to avoid memory leaks. This is a nice little helper method.
