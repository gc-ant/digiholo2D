sharedptr< std::vector< tile > > abstract_gradient_calculator::get_gradient_tile ( sharedptr< tile >  t)
virtual

Get the gradient of a given tile in the tile-format. First element of the vector will be a tile representing the x gradient and second element will be a tile representing the y gradient. This method is for debug purposes mostly since it will call the abstract get_gradient_floats method of the instance.
