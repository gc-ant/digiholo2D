Detailed Description

merges individually unwrapped tiles on the SRNCP algorithm from the Harraez et al Paper. The reliability measure for tiles if very different from the measure for single pixels though.

void srncp_tile_merger::merge_tiles ( tesselated_image *  t)

This method merges tiles from a tesselated wrapped phase image to an unwrapped image. These tiles alreasy need to be unwrapped individually and are then unwrapped with respect to each other to form an unwrapped phase map.
