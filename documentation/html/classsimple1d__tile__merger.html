Linear unwrapper based on 1d unwrapping of blocks.The booleans are useful if the image has discontinuties on the edges.

Parameters
left2rightSpecify the horizontal direction of merging: true: Start left and head right. false: right to left.
top2bottomSpecify the vertical direction of merging: true: Start from the top to the bottom edge. false: bottom to top.
start_horizontalSpecify order of merging: true: First merge horizontal, then vertical. false: First vertical, then horizontal

void simple1d_tile_merger::merge_tiles ( boost::shared_ptr< tiled_image >  ti)
virtual

Merge tiles of a tiled image. The tiles have to be unwrapped inside themselves. This method unwraps the tiles with respect to each other. The phase discontinuity between each tile must be integer multiples of 2PI.
